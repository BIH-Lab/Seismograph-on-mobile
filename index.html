스마트폰 지진계

스마트폰 하나로 지진동을 측정하고 분석하는 웹 플랫폼

01

지진파 색으로 보기

스마트폰 센서로 진동을 측정하고,
진폭에 따라 화면 색이 변합니다.

→
02

지진파 기록하기

GPS 좌표와 함께 Z축 가속도를
기록하고 CSV로 저장합니다.

→
03

주파수 분석 (스펙트로그램)

실시간 FFT로 진동 신호의
주파수 성분을 색상으로 표시합니다.

→
04

주시곡선 탐사 (굴절법)

여러 스마트폰 기록으로 주시곡선을 그리고
표층 두께를 계산합니다.

→
05

주시곡선·진원 찾기

P파·S파 도달 시간 및 GPS 위치로
진원지를 역산합니다.

준비중
GitHub 토론

사용법

  1. 스마트폰 브라우저 (iOS: Safari / Android: Firefox)로 접속하세요.
  2. Activity 카드를 선택하세요.
  3. 측정 시작 버튼을 누르면 센서 권한을 요청합니다.
  4. 권한을 허용하면 실시간 측정이 시작됩니다.

※ iOS는 Safari, Android는 Firefox를 권장합니다. (Chrome은 센서 정밀도 제한)
