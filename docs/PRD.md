# PRD.md — Product Requirements Document

## 제품 목적
스마트폰 내장 가속도 센서를 활용해 지진동 데이터를 실시간 수집하고,
웹 플랫폼에서 시각화·통계 처리·다운로드까지 가능한 연구/교육용 계측 시스템 구축.
앱 설치 없이 브라우저만으로 동작하는 것이 핵심 차별점.

## 사용자
- 연구자 (데이터 수집 및 분석 목적)
- 교사/학생 (지구과학 교육 현장 활용)
- 관리자 (플랫폼 운영)

## 핵심 기능 요구사항

### Cycle 1 — MVP
| 기능 | 설명 |
|------|------|
| 센서 권한 요청 | iOS/Android 브라우저에서 DeviceMotion 권한 처리 |
| 실시간 데이터 수집 | X·Y·Z축 가속도 + 타임스탬프 수집 |
| 실시간 그래프 | 수집 중 실시간 파형 표시 |
| 측정 시작/정지 | 버튼으로 측정 제어 |
| CSV 다운로드 | 수집된 데이터를 CSV 파일로 저장 |

### Cycle 2 — 사용자 관리
| 기능 | 설명 |
|------|------|
| 회원가입/로그인 | 이메일 기반 인증 |
| 세션 관리 | 측정 세션별 저장 및 이력 조회 |
| 데이터 서버 저장 | 측정 데이터를 DB에 영구 저장 |

### Cycle 3 — 분석 및 시각화
| 기능 | 설명 |
|------|------|
| 기본 통계 | 최대값·최소값·RMS·샘플링 레이트 자동 계산 |
| FFT 주파수 분석 | 주파수 스펙트럼 시각화 |
| 세션 비교 | 여러 측정 결과 오버레이 비교 |

### Cycle 4 — 고도화
| 기능 | 설명 |
|------|------|
| 실험 그룹 관리 | 연구자가 그룹 생성 후 참가자 초대 |
| 데이터 일괄 다운로드 | 그룹 전체 데이터 ZIP 다운로드 |
| 관리자 대시보드 | 사용자·세션 현황 모니터링 |

## 비기능 요구사항
- 모바일 퍼스트 반응형 UI
- HTTPS 필수 (센서 API 요구사항)
- iOS Safari / Android Chrome 크로스 브라우저 지원
- 센서 샘플링 레이트 : 목표 100Hz
- 데이터 정밀도 : 소수점 6자리 이상 유지
