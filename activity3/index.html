스펙트로그램

시간의 흐름에 따라 각 주파수에 담긴 에너지를 색상으로 표시합니다.

  1. X축 — 시간 (오른쪽=현재, 왼쪽=과거)
  2. Y축 — 주파수 (Hz)
  3. 색상 — 검정→파랑→초록→노랑→빨강 순으로 에너지 증가
  4. 지진파 도달 시 특정 주파수 대역에서 밝은 색이 갑자기 나타납니다

FFT 크기: 256, 홉: 26샘플 (~97.5% 오버랩), 표시 범위: 30초

PSD (Power Spectral Density)

각 주파수에 얼마나 많은 에너지가 담겨 있는지 측정합니다.

  1. X축 — 주파수 (로그 스케일, Hz)
  2. Y축 — 에너지 밀도 (dB re 1 m²/s⁴/Hz)
  3. 여러 시간 구간을 평균내어 안정적인 스펙트럼을 계산합니다 (Welch 방법)
  4. 뾰족한 봉우리: 특정 주파수의 강한 신호 (기계 진동, 지반 공진 등)

FFT 크기: 1024 (Δf ≈ 0.1 Hz), Hann 윈도우, DC 제거 적용

HVSR (수평/수직 스펙트럼 비율)

수평 진동과 수직 진동의 스펙트럼 비율로 지반 공진 주파수를 추정합니다.

  1. X축 — 주파수 (로그 스케일, Hz)
  2. Y축 — H/V 비율
  3. H/V 비율이 높은 주파수 = 지반 공진 주파수 (f₀) 후보
  4. f₀ ≥ 2.0 이상의 뚜렷한 피크일 때 신뢰도 높음
  5. 3축 CSV 파일 (acc_x, acc_y, acc_z) 이 필요합니다

파일 모드: Konno-Ohmachi 스무딩 (b=40), 50% 오버랩, 정상성 필터 적용
